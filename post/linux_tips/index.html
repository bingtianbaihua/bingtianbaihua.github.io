列举我常用的linux命令

2016-09-05

Contents

今天写写自己常用的linux命令，也是建立在能快速解决问题的基础上，不在于越多越好。 当然我个人非常欣赏能在纯命令行下工作的人，自己也逐渐在这个方向尝试。

  1. 系统资源性能查看 > top 查看cpu，内存，磁盘等状况 iotop 查看io的情况 iostat 同上 netstat 查看网络情况，查看端口的话，用netstat -apn|grep 端口号， 是否被占用 lsof -i:端口号 ，也是查看是否被占用的情况 ps 看系统进程情况，我经常是 ps -ef|grep “进程字符串”查看tomcat情况 kill 杀死进程

  2. 文件处理情况 > mkdir 新建文件夹 rmdir 删除空文件夹 rm 删除一个文件，递归删除文件夹，我常用 rm -rf 文件夹 find 查找文件 mv 移动或者重命名文件 ln -s 创建软链接，非常好用，因为对于用中文命名的文件夹，cd到那边是极其麻烦的 tail -f 查看文件变化 more/less/cat/tac 都是查看文本的命令

  3. 常用的移动，命令行和打开操作等 > cd 到某个目录下，直接cd回到home目录下 ls和ll， 一个列出文件，另外一个还列出权限等 ssh跳到目的主机 scp 复制文件到目的主机 awk，非常厉害的一个命令，略知一二 pwd 查看当前目录 alt+f4 关闭 alt+f2 查找应用 f11 全屏 ctrl+u 清除整个命令行 ctrl+W 清除以空格分割的一个命令行部分 ctrl+l 清屏 ctrl+shift+v 粘贴 ctrl+alt+t 新建一个终端 ctrl+t 新建一个终端标签页

  4. 文件权限 >chmod chown
    这两个用的不多，但是要知道

5.解压缩和压缩 > 我常用的两个 tar -zcvf 源文件 目的文件， 压缩文件 tar -zxvf 源文件 目的地址 ，解压文件， -C到指定目录下 当然还有zip和unzip，我用的不多。 jar -xvf war包 ，部署java war包用的

好了，我暂时能想到的就是这些了，可能有不少遗漏的，后续想到了再慢慢补。用好linux，的确对自己工作效率的提高是有很明显的效果的，希望自己坚持在linux环境下工作！！！
